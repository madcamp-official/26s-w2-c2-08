COURSE_CREATE_PAGE · PRE-T-01

한 학기 수업방을 시작합니다.

로그인한 사용자는 누구나 Course를 만들 수 있습니다. 생성자는 이 Course에서 교수자가 되며 계정 전체의 역할은 바뀌지 않습니다.

Course 기본 정보

학생들이 알아볼 수 있는 과목명과 표시용 학기를 입력하세요.

구체적인 최대 길이는 아직 제품 정책에서 정하지 않았습니다.

공백이 아닌 과목명을 입력해주세요.

현재 API는 구조화된 값이 아닌 표시용 문자열로 받습니다.

공백이 아닌 학기를 입력해주세요.

취소

생성 후 이렇게 동작해요

  • 생성자는 해당 Course의 교수자로 등록됩니다.
  • 서버가 영문 대문자 6자리 참여 코드를 발급합니다.
  • 참여 코드는 교수자 화면에서만 표시됩니다.

참여 코드는 구분자 없는 영문 대문자 6자리이며 만료되지 않습니다. Course를 처음 만든 교수자는 나중에 새 코드를 발급해 기존 코드를 즉시 폐기할 수 있습니다.

Course와 교수자 멤버십을 생성하는 중입니다.

✓

알고리즘 Course를 만들었습니다

김도현님은 이 Course에서 교수자입니다. 아래 구분자 없는 영문 대문자 6자리 코드를 학생에게 공유하세요. 코드는 만료되지 않으며 생성자인 김도현님만 나중에 새 코드로 교체할 수 있습니다.
