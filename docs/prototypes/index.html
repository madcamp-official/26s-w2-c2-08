GOAL · Static prototype

화면 흐름을 한곳에서 검토합니다.

GOAL 1차 MVP의 역할별 화면, 처리 상태와 연결 흐름을 확인하는 인덱스입니다. 현재 카드들은 구현 순서를 보여주며, 화면이 준비되면 직접 열 수 있는 링크로 전환됩니다.

1. 서비스 진입과 대시보드

MAIN_PAGE 비로그인 메인 준비 중
MAIN_PAGE_AUTH 로그인 후 대시보드 준비 중
MY_INFO_PAGE 내 정보 준비 중

2. Course 생성과 참여

COURSE_CREATE_PAGE Course 생성 예정
COURSE_JOIN_PAGE Course 참여 예정
COURSE_PAGE_* 역할별 Course 예정

3. class 준비·실시간·기록

CLASS_CREATE_PAGE class 생성과 PDF 예정
LIVE_CLASS_PAGE_* 역할별 실시간 class 예정
ENDED_CLASS_PAGE_* 정리 중·완료 기록 예정

이 페이지와 연결 화면은 제품 검토용 정적 프로토타입입니다. 실제 계정, 수업 데이터, STT 또는 AI 결과를 사용하지 않습니다.
