    실제 API 경계

    이 Prototype에서는 아래 요청을 실행하지 않습니다.

    • GET /courses/{course_id}교수자 역할·현재 LIVE 확인
    • POST /courses/{course_id}/sessionsREADY Session 생성
    • GET /sessions/{session_id}재진입 시 Session 상태와 이동 경계 확인
    • POST /sessions/{session_id}/materialsPDF 한 개당 한 요청, 202 + Material·Job
    • GET /sessions/{id}/materials · GET /materials/{id} · GET /sessions/{id}/jobs목록·Material·MATERIAL_PROCESSING 상태
    • POST /jobs/{job_id}/retry같은 Job 행의 attempt 증가
    • POST /sessions/{session_id}/startREADY → LIVE
