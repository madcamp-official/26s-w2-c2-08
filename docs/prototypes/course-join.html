참여 코드로 수업방에 들어갑니다.

참여에 성공하면 이 Course에서 학생이 됩니다. 다른 Course에서 가진 역할에는 영향을 주지 않습니다.

참여 코드 입력

교수자가 Course 화면에서 공유한 코드를 그대로 입력하세요.

앞뒤 공백을 제거하고 영문은 대문자로 바꿔 확인합니다. 코드의 길이와 사용 문자 정책은 아직 미정입니다.

공백이 아닌 참여 코드를 입력해주세요.

취소

Course별 역할

  • 코드 참여자는 해당 Course의 학생으로 등록됩니다.
  • 이미 학생이면 같은 멤버십으로 안전하게 이동합니다.
  • 기존 교수자 역할은 학생으로 덮어쓰지 않습니다.

유효하지 않은 코드에는 Course 존재 여부나 만료 여부를 과도하게 노출하지 않는 공통 안내를 사용합니다.

참여 코드와 Course 접근 가능 여부를 확인 중입니다.
