# ToBeDetermined 기획안

## 1. 프로젝트 개요

**ToBeDetermined**는 오프라인 수업을 위한 실시간 학습 보조 및 복습 서비스이다. 수업 중에는 학생이 익명으로 질문하고 AI 요약을 통해 놓친 내용을 따라갈 수 있으며, 수업 후에는 강의 자료와 질문·답변 기록을 기반으로 AI 튜터와 복습할 수 있다.

### 목표 사용자

- 대학교 교수, 강사 등 다수의 학습자를 가르치는 교수자
- 오프라인 수업에 참여하는 학생

## 2. 해결하려는 문제

### 학생

- 공개적으로 질문하기 부담스럽다.
- 수업 중 놓친 내용을 바로 따라가기 어렵다.
- 복습할 때 교수자가 실제로 설명한 내용을 찾기 어렵다.
- 다른 학생들이 어떤 부분을 어려워했는지 알기 어렵다.

### 교수자

- 학생들이 어느 지점에서 막혔는지 실시간으로 파악하기 어렵다.
- 비슷한 질문이 반복되어 수업 흐름이 끊긴다.
- 수업이 끝난 뒤 질문과 답변을 별도로 정리하기 번거롭다.

## 3. 핵심 해결 방식

교수자는 한 학기 단위의 수업방(Course Room)을 만들고, 매 수업마다 강의 세션(Lecture Session)을 생성한다. 각 세션에는 강의자료, 질문, 답변 및 AI 정리 결과가 축적된다.

학생은 수업 중 익명 질문과 실시간 기록을 활용하고, 수업 후에는 해당 세션의 자료와 질문·답변을 근거로 AI와 대화하며 복습한다.

```text
수업방 (한 학기)
├── 03/05 강의 세션
│   ├── 강의자료 PDF
│   ├── 강의 기록(Transcript)
│   ├── 학생 질문 및 반응
│   ├── 교수자 답변
│   ├── AI 요약
│   └── 복습용 AI 채팅
├── 03/12 강의 세션
└── 03/19 강의 세션
```

## 4. 주요 기능

### 수업 전

**교수자**

- 한 학기 수업방 및 수업 코드 생성
- 오늘의 강의 세션 생성
- 강의자료 PDF 업로드
- 수업 시작

**학생**

- 수업 코드로 수업방 참여
- 오늘의 강의 세션 입장

### 수업 중

**교수자**

- 학생의 익명 질문 실시간 확인
- AI가 유사 질문을 묶은 질문 클러스터 확인
- ‘나도 궁금해요’ 반응이 많은 질문 우선 확인
- 선택한 질문에 음성 또는 텍스트로 답변

**학생**

- 익명 질문 작성
- 다른 질문에 ‘나도 궁금해요’ 반응
- 강의자료와 현재까지의 수업 기록을 바탕으로 AI에게 질문
- “방금 내용 요약해줘”, “놓친 부분을 설명해줘” 등의 따라잡기 요청

### 수업 후

- 해당 날짜의 강의 기록 페이지 자동 생성
- 강의자료, 수업 기록, 질문 및 교수자 답변 저장
- AI가 강의 핵심 내용과 질문·답변을 요약
- 해당 강의 기록을 기반으로 한 복습용 AI 채팅 제공

## 5. 사용 시나리오

1. 교수자가 수업방을 만들고 학생에게 수업 코드를 공유한다.
2. 교수자가 오늘의 강의 세션을 생성하고 강의자료 PDF를 업로드한다.
3. 학생이 수업 코드로 참여하고 오늘의 세션에 입장한다.
4. 수업 중 학생은 익명 질문을 작성하거나 기존 질문에 반응한다.
5. 교수자는 실시간 질문 목록과 AI가 묶은 유사 질문을 확인한다.
6. 교수자가 주요 질문에 답변하면 질문과 답변이 함께 기록된다.
7. 수업 종료 후 강의 기록 페이지가 자동 생성된다.
8. 학생은 저장된 자료와 질문·답변을 확인하고 AI 튜터와 복습한다.

## 6. 1차 MVP 범위

### 필수 기능

1. 교수자 수업방 생성 및 수업 코드 발급
2. 오늘의 강의 세션 생성
3. 강의자료 PDF 업로드
4. 학생의 수업 코드 입장
5. 익명 질문 작성 및 실시간 반영
6. 질문에 ‘나도 궁금해요’ 반응
7. LLM 기반 유사 질문 클러스터링
8. 교수자 답변 기록
9. 수업 종료 후 강의 기록 페이지 생성
10. 해당 강의의 질문·답변·자료를 기반으로 한 AI 채팅

### 후순위 기능

- 실시간 음성 인식(STT) 및 고품질 Transcript
- 강의 음성 녹음 파일 저장
- 교수자 음성과 학생 음성 자동 분리
- 수업 전체 자동 챕터링
- 한 학기 전체 기록 통합 검색 및 AI 채팅

## 7. MVP 핵심 가치

1차 MVP는 완전한 실시간 음성 처리보다 **익명 질문 수집, 유사 질문 정리, 수업 기록 기반 복습**의 효용을 검증하는 데 집중한다.

- 학생은 부담 없이 질문하고 다른 학생의 어려움도 함께 확인할 수 있다.
- 교수자는 수업 흐름을 크게 끊지 않고 질문의 우선순위를 파악할 수 있다.
- 수업이 끝나면 질문과 답변이 자동으로 축적되어 복습 자료가 된다.
